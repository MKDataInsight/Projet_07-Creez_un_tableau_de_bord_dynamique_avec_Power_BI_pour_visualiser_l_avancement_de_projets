## Scénario

![esndata](/assets/esndata.png)

![sanitorial](/assets/sanitorial.png)

Je suis Data Analyst chez ESNDATA, en mission chez le client Sanitorial, une entreprise internationale qui fabrique et commercialise des soins bucco-dentaires.

Pour le Project Management Office, je dois élaborer un tableau de bord interactif permettant de suivre l'avancement des projets, afin que l'équipe puisse contrôler les performances et prendre les décisions opportunes pour limiter les retards.

## Objectifs et compétences
- **Objectif** : Produire un reporting en analysant les visualisations pour faciliter les décisions

**Compétences développées** : 

- Identification des enjeux du client
- Collecte des user stories
- Préparation et la transformation des données
- Développement des visualisations et enfin la présentation au client

## Technologies utilisées
- **Power BI** : Transformation des données et élaboration du tableau de bord

## Structure du projet
Le projet est organisé comme suit :
- **Donnees_initiales** : Données relatives aux projets d'ESNDATA.
- **Livrables** : Contient le tableau de bord interactif.
- **Besoins_client.pdf**, **ProductStrategyCanvas.pdf** : Documentation technique pour la gestion de ce projet
